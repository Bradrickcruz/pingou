# 🧩 Tarefa: Migrar painel web para Tailwind CSS

## 💡 Ideia
Substituir o CSS atual (vanilla CSS com variáveis) por Tailwind CSS para melhorar a manutenibilidade e velocidade de desenvolvimento.

## 🎯 Objetivo
Utilizar Tailwind CSS em todo o frontend do painel web.

## 📦 Escopo inicial
- Instalar e configurar Tailwind no projeto web
- Converter index.css e App.css para classes Tailwind
- Migrar componentes em src/components/

## 🚧 Fora de escopo
- Criar novos componentes ou funcionalidades
- Mudar design/visual além da migração técnica

## 🔍 Contexto
O projeto web atualmente usa CSS vanilla com variáveis em `src/index.css` e `src/App.css`, além de styles inline nos componentes JSX.

## ⚠️ Observações
- Manter o tema atual (cores claras/escuras)
- Verificar se há conflicts com styles inline existentes
